JavaScript là gì?

JavaScript là một ngôn ngữ lập trình kịch bản (scripting language) được sử dụng chủ yếu để tạo trang web tương tác. JavaScript chạy trên trình duyệt web và cho phép developer thêm động lực và tương tác vào trang web tĩnh.

Lịch sử phát triển

JavaScript được tạo ra bởi Brendan Eich tại Netscape vào năm 1995. Ban đầu được gọi là Mocha, sau đó đổi thành LiveScript, và cuối cùng là JavaScript. Mặc dù có tên “Java” nhưng JavaScript không liên quan đến Java.

Đặc điểm nổi bật

  • Client-side scripting: Chạy trên trình duyệt của user
  • Interpreted language: Không cần compile
  • Dynamic typing: Kiểu dữ liệu được xác định tại runtime
  • Event-driven: Phản ứng với các sự kiện của user
  • Asynchronous: Hỗ trợ lập trình bất đồng bộ

Ứng dụng của JavaScript

1. Web Development

  • Tạo tương tác trên trang web
  • Validation form
  • Animation và effects
  • AJAX calls

2. Server-side Development (Node.js)

  • Xây dựng API
  • Real-time applications
  • Microservices

3. Mobile App Development

  • React Native
  • Ionic
  • Cordova

4. Game Development

  • HTML5 games
  • WebGL

Cú pháp cơ bản

// Biến
let name = "JavaScript";

// Function
function greet(name) {
    return "Hello, " + name;
}

// Event listener
document.getElementById("btn").addEventListener("click", function() {
    alert("Button clicked!");
});

JavaScript là ngôn ngữ không thể thiếu trong phát triển web hiện đại, với ecosystem khổng lồ gồm các framework và library như React, Vue, Angular, Node.js.
